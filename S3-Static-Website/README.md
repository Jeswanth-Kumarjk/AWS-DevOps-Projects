# S3 Static Website Hosting on AWS

## Project Overview
This project demonstrates hosting a static website using Amazon S3.

## What I Implemented

- Created an S3 bucket
- Enabled static website hosting
- Uploaded HTML and CSS files
- Configured bucket policy for public access
- Accessed website using S3 endpoint URL

## Services Used

- Amazon S3
- IAM
- HTML
- CSS

## Outcome

Successfully hosted a static website using Amazon S3 without using any server.
